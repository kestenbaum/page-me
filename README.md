# Portfolio | Oleksii Vovnenko

A modern, high-performance developer portfolio built with **React** and **TypeScript**. The project has been fully refactored for maximum efficiency, featuring serverless email integration, optimized bundle size, and automated code styling.

## 🚀 Tech Stack

*   **React 18** — UI library for building interactive components.
*   **TypeScript** — Strict typing for enhanced reliability and scalability.
*   **CSS Modules** — Scoped styling to prevent global namespace pollution.
*   **Framer Motion** — Production-ready animations for a polished feel.
*   **EmailJS** — Serverless email integration to handle contact form submissions.
*   **React Hot Toast** — Elegant notifications for real-time user feedback.
*   **i18next** — Localization support for multilingual content.

## ✨ Key Features

- **Serverless Contact Form**: Direct communication via EmailJS with custom hook integration and `ref`-based validation.
- **Performance Optimized**: Removed heavy dependencies (Axios, React Query) to ensure lightning-fast load times and a minimal footprint.
- **Advanced UX**:
  - **Dynamic Header**: Fixed navigation that adapts its style upon scrolling.
  - **Smart Scroll-to-Top**: A discreet button that appears only after scrolling past the first section.
  - **Smooth Anchors**: Seamless navigation between sections for a fluid experience.
- **Developer Experience (DX)**:
  - **ESLint**: Configured to catch unused code and potential bugs early.
  - **Prettier + Import Sorting**: Automated formatting with `@trivago/prettier-plugin-sort-imports` for a consistent codebase.

## 🛠 Installation & Setup

1.  Clone the repository:
    ```bash
    git clone [https://github.com/vovnenko/portfolio-react-app.git](https://github.com/vovnenko/portfolio-react-app.git)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm start
    ```
4.  Build for production:
    ```bash
    npm run build
